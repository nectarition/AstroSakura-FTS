サークル参加ガイド

2024年 1月 21日 更新

サークル参加条件

参加資格

  • サークル代表者が義務教育を修了していること。
  • 成人向けの頒布物を用意するサークルはサークル代表者が18歳以上であること。
  • 双葉湊音に関連した作品を当日１点以上配布できること。
    • 創作物の頒布に関しては、各キャラクターのライセンスおよび音声利用のガイドラインをご確認ください。

募集スペース数

イベント全体で直接参加50スペース

参加費用

  • 1スペース (机半分(幅90cm x 奥行き45cm x 高さ70cm), サークル通行証 2枚): 4,000円
  • 2スペース (机全面(幅180cm x 奥行き45cm x 高さ70cm), サークル通行証 4枚): 8,000円

青春18割

  • 開催日時点でサークル代表者が高校生世代の方(就学の有無は不問)は、青春18割としてサークル参加費のキャッシュバックを行います。
  • 当日のサークル受付時に学生証や身分証をご提示いただくと、1スペースあたり1,000円をご返金いたします。

青春18割の対象者

  • イベント開催日にサークル代表者が高校生世代（2006年4月2日から2009年4月1日生まれの方）であること。
  • イベント当日にサークル代表者が参加ができること。欠席の場合は対応できかねます。

頒布物・展示物の注意事項

禁止頒布物

  • 刑法175条に定める「猥褻図画類」を含む物品
  • 古物や法人発行の物品
  • 各キャラクター利用ガイドラインに反した内容を含む物品
  • 手作り既製品を問わず、不特定多数に対する食品（DECOチョコ／企業作成による個別包装されたものは可能）
  • 日本国における各種法令に反する物品

奥付について

頒布する作品の奥付には、下記の内容を 必ず 記載してください。

  • 発行者名(サークル名, ペンネーム可)
  • 発行日
  • 連絡先(メールアドレス, または電話番号)
    • ※SNSなど連絡の際に登録が必要な連絡先はNG
  • 印刷所名(自家印刷やコンビニコピーの場合は省略可)

その他

  • 会場の電源は使用できません。
  • 音楽や動画を頒布する際はオリジナルまたは製作者からの許諾が得られている素材を使用したものに限ります。
  • イベント当日発行の新譜、新刊等は見本誌として回収致します。
  • グッズ等の提出は任意としますが、内容確認のために写真撮影や出力をさせていただく事があります。

サークル申込方法

注意事項

締切

  • 1次締切: 2024年 2月 17日 (土)
  • 最終締切: 2024年 3月 31日 (日)

※申込状況によって1次で満了の可能性がございます。
また、申込多数の場合は、抽選とさせていただく可能性がございます。

キャンセルについて

  • 原則として、申込後のキャンセルはできません。
  • 抽選、申込内容不備等により落選の場合は、手数料を差し引いて返金させていただきます。
  • 自己都合でのキャンセル・ご返金は出来かねますので予めご了承ください。

中止について

  • 天災や会場側の要請等の不可抗力によりイベントの開催を中止する場合があります。
  • この場合のサークル参加費は手数料を差し引いて返金します。

サークル参加に必要なもの

  • サークルカットの作成
  • Sockbaseからイベントへの申込、 サークルカットの提出
  • サークル参加費の支払い

申し込みの流れ

サークルカットの用意

  • サークルカットは、テンプレートを利用して作成してください。
  • テンプレートを最も上のレイヤーに設定し、PNGファイルで書き出してご提出ください。
  • サークルカットの中に必ずサークル名を入れてください。
  • サークルカットでの過激・露骨な性描写等はおやめください。不特定多数の方の閲覧が可能なためご配慮をお願いいたします。
  • フルカラー・モノクロのどちらでご作成いただいても問題ございませんが、カタログにはモノクロで掲載予定です。

サークルカットテンプレート

申し込み情報の入力

申し込みページ

上のボタンから申込ページを開き、サークル情報の入力を行ってください。
プチオンリーコードの入力は不要です。

隣接希望申請を行う場合、まず一方のサークルが申込を行って表示された申し込みIDを、もう一方のサークルに伝え「合体希望サークル 合体申し込みID」の欄に入力してください。

  • ３つ以上のサークルによる隣接希望は別途「通信欄」にその旨をご記入ください。(申し込みフォームの「合体希望サークル 合体申し込みID」からは受け付けられません)
  • 別のイベントに申し込んだサークルとの隣接希望は受け付けられません。予めご了承ください。

サークル参加費のお支払い

  • サークル参加費のお支払いには、クレジットカード(オンライン決済)、または銀行振込がご利用いただけます。
  • サークル申込受付終了日までに オンライン決済、銀行振込にてお支払いをお願いいたします。
  • 銀行振込は口座をお持ちでなくてもご利用できます。詳しくはご利用の金融機関のウェブサイトをご確認ください。
  • 申し込みページを閉じた後に決済を行う場合、マイページにログインし「決済履歴」から対象の支払い先を選択し、決済を行ってください。
  • 銀行振込を選択した場合は、お申し込み時、または「決済履歴」に表示される6ケタの「補助番号」を、ご依頼人名義の先頭に付けて振込手続きを行ってください。
    • 例： 283710ﾌﾀﾊﾞﾐﾅﾄ

当日の流れ

後日、お申し込みの際にご入力いただいたメールアドレス宛にサークル参加要項を送付いたしますので、必ずご確認ください。
